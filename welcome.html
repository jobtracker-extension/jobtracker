Bienvenue sur JobTracker

Sauvegardez vos annonces, suivez vos candidatures et archivez
les pages hors ligne — tout reste sur votre machine.

Épinglez l'extension à votre barre d'outils

Cliquez sur ··· (Extensions) dans la barre → trouvez JobTracker → cliquez l'icône 📌 pour l'épingler. Vous pourrez ainsi ouvrir le panneau en un clic.

1

Ouvrez le side panel sur une annonce

Cliquez l'icône JobTracker dans la barre d'outils sur n'importe quelle page d'annonce.

2

Pointez les éléments avec le viseur

Cliquez l'icône viseur à côté d'un champ, puis cliquez l'élément sur la page. L'URL est récupérée automatiquement.

3

Sauvegardez — une capture est archivée

Un snapshot complet de la page est créé automatiquement (CSS, images, fonts inlinés). Consultable hors ligne même si l'annonce est retirée.

4

Suivez depuis le tableau de bord

Cochez En ligne, Par mail ou Refus — la date s'enregistre au clic. Exportez en CSV ou JSON avec snapshots inclus.

C'est parti

Toutes vos données restent sur votre machine — aucun serveur externe.
